# Image to PDF Converter

A simple Python script to convert multiple images into a single PDF file. Supports `.jpeg`, `.jpg`, and `.png` formats.

## Features

- Automatically sorts images by filename (numeric order if filenames are numbers).  
- Supports JPEG, JPG, and PNG images.  
- Generates a single PDF from multiple images.  

## Usage

1. Place your images in the same folder as the script.  
2. Run the script:

```bash
python image_to_pdf.py
